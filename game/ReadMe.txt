作品説明欄:
作成期間:4ヶ月
自動車工学を1ヶ月独学で勉強して作成した自動車シミュレータになります。
エンジンやハンドルの仕組みを再現したいと思いエンジンの細かい所を再現しました。
エンジンの動かしを４ストロークエンジンを採用し、
燃料による過給・エンジンのピストンの動きからクランクシャフトを通すところまで作りました
またハンドルからタイヤに伝わるまでの過程も勉強し作りこめる部分をとことん追求作品です。

苦労した点:
ゲーム性におく場合エンジンはいじることはあまり無いのですが、リアルの再現をするべく
４ストロークエンジンや燃料・排気量などを調べる必要があった為エンジン作成に時間がかかりました。
またエンジンの動きがわからないところは自動車工場に行って見に行ったり教わったりしていました。
